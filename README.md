# Todo-web

API simples feita para gerenciar listas e itens

## Tecnologias utilizadas

-   Laravel 12
-   Pest
-   Sanctum
-   Scramble
-   SQLite

## Documentação

Para acessar a documentação da API, acesse a rota `docs/api` (caso esteja rodando local, o link seria `http://localhost:8000/docs/api`).

> [!IMPORTANT]
> Este projeto possui somente a parte da API, não há front-end!
